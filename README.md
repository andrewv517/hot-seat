# Hot Seat
A jackbox style implementation of the card game Hot Seat using
Vue.js and socket.io.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
